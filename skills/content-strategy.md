# Content Strategy

Use this skill when turning a website brief into usable page copy.

## Goals

- Make the hero headline concrete and easy to scan.
- Turn feature lists into benefits, not vague claims.
- Shape CTA labels so they match the user's intent.
- Provide SEO title and description that feel like the page, not boilerplate.

## Output Rules

- Keep copy specific to the brief.
- Prefer short, direct sentences.
- Make section copy useful to both human readers and the Vue generator.
